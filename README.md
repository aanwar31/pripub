# pripub
```sh
pripub private or public, anonymously chat app and chrome extension for beginners  (2014)
nodejs + socket.io
without express, redis, sql
```


##install
```sh
npm install
```

##start
```sh
node server.js
```

##features
```sh
sending image
online user counter
totally messages counter
changeable nicks
@nick sound alert
chrome extension uses pripub
```
